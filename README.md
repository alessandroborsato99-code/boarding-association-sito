# Board Association — Sito (mockup)

Mockup statico multi-pagina per Board Association (Verona) — scuola e community di snowboard, sci, skate, surf e wake. Parte del gruppo Detour / Slide (Generation S.r.l.).

## Pagine
- `Home-BoardAssociation.html` — home (hero, manifesto, corsi, noleggio, uscite, magazine, crew, prima esperienza, contatti)
- `Corsi-BoardAssociation.html` — corsi, listino lezioni, pacchetti, livelli, tessere, winter tour
- `Noleggio-BoardAssociation.html` — listino noleggio attrezzatura
- `Magazine-BoardAssociation.html` — magazine / uscite / news
- `ChiSiamo-BoardAssociation.html` — chi siamo, manifesto, negozi
- `Team-BoardAssociation.html` — maestri, assistenti e guide
- `Contatti-BoardAssociation.html` — contatti, form, mappa

## Stack
HTML + CSS statico (single-file per pagina, stile inline). Nessuna build. Aprire `index.html` o servire la cartella:

```
python3 -m http.server 8765
```

Font: Archivo + Space Mono (Google Fonts). Immagini in `assets/`.
